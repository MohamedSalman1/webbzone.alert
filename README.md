# Webbzone Alert

This js plugin provide features to alert, confirm and inform dialogs to user irrespective of using traditional browser alerts.

## Available Features

- Alert Box
- Confirm Box
- Inform Box
- Flash messages (Success, Danger, Info)
- Loaders
